# LR4_part1_Goshko
Task Tracker Гошко Я.И. — веб-приложение на Django, разработанное в среде Replit . Приложение реализует отображение списка задач пользователя через HTML-шаблон и Django-представление. 

Проект включает:
приложение tasks;
маршрутизацию URL;
Django-представление;
HTML-шаблон index.html;
скрипт развертывания deploy.sh.

Пользователь может изменять:
название приложения;
список задач;
оформление HTML-страницы;
Bootstrap-компоненты интерфейса;
содержимое шаблона index.html;
маршруты URL;
логику представления views.py.

Для запуска необходимо выполнить:
1. chmod +x deploy.sh
2. ./deploy.sh


Скрипт автоматически выполняет миграции и запускает Django-сервер на порту 3000. Проект может быть развернут через GitHub-репозиторий в новой реплике Replit.
